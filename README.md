# Analyse des Tournois de Tennis ATP

Modélisation des joueurs et de leurs matchs sous forme de graphes pour extraire des insights sur les performances, les classements et les relations entre joueurs.

## Technologies utilisées
- **GraphX** : Pour la modélisation et l'analyse des graphes de joueurs et de matchs
- **GraphFrames** : Pour des requêtes avancées et des patterns de graphes

## Vue d'ensemble
Ce projet analyse les données des tournois ATP (2020-2022) en représentant les joueurs comme des nœuds et les matchs comme des arêtes, permettant d'identifier les patterns, les rivalités et les performances clés.
